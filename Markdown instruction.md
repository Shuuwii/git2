# Инстуркция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом нужно обрамить его звездочками(*.) Например,   *вот так*.

Чтобы выделить текст полужирнымБ необходимо обрамить его двойными звездочками(**).
Например, **вот так**.
